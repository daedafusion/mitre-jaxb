# mitre-jaxb

JAXB Libraries for stix, maec & cybox. The following versions of the schemas are used:

    cybox v2.1
    maec  v4.1
    stix  v1.1.1

## Maven

```xml
<dependency>
    <groupId>com.daedafusion</groupId>
    <artifactId>mitre-jaxb</artifactId>
    <version>1.1</version>
</dependency>
```
